Intro

Hey! This is Rajea, a Software Engineer based in Birmingham, England. Coming from a design background, I have a passion for creating beautiful websites and web applications that load fast, run across all devices and are accesible to everyone with an internet connection. Some of my work.

I'm proficient in both frontend and backend development. I have extensive knowledge in HTML, CSS, Javascript, Node.Js, MongoDB, and React.

I love working with small businesses and individuals to create a web presence for them. Whether that is creating a website from the ground up, or updating an existing presence for the mobile age, I do it all :) How can I help you today?
